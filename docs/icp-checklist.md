# ICP 备案准备清单（企业备案）

## 阿里云ECS购买配置

| 项目 | 推荐配置 |
|------|---------|
| 地域 | 上海 / 杭州 / 北京 |
| 实例 | 1核1G（最低配即可，约¥100/月）|
| 系统盘 | 40GB SSD |
| 操作系统 | Ubuntu 22.04 LTS 64位 |
| 带宽 | 1Mbps（备案期间足够）|
| 时长 | 至少3个月（备案要求）|

## 备案所需材料

### 必备材料

1. **营业执照**（原件彩色拍照/扫描）
   - 必须包含完整边框
   - 剩余有效期 ≥ 3个月
   - 不能是复印件、黑白扫描件
   - 建议文件大小：500KB - 2MB

2. **法定代表人身份证**（原件彩色拍照/扫描）
   - 正反面都需要
   - 剩余有效期 ≥ 3个月
   - 必须包含完整边框

3. **网站负责人身份证**（原件彩色拍照/扫描）
   - 如果网站负责人 ≠ 法定代表人，需要额外提供**授权书**
   - 剩余有效期 ≥ 3个月

4. **域名证书**
   - 域名持有者必须与营业执照上的企业名称完全一致
   - 域名有效期 ≥ 6个月
   - 备案期间关闭域名隐私保护

### 可选材料（根据管局要求）

5. **授权委托书**（如网站负责人非法定代表人）
6. **网站建设方案书**（广东、湖北等省份，域名过多时）
7. **前置审批承诺书**（如涉及医疗、新闻等行业）

## 域名检查清单

- [ ] 域名在阿里云账号下（或已转入阿里云）
- [ ] 域名已完成实名认证
- [ ] 域名实名认证主体 = 营业执照企业名称
- [ ] 域名有效期 > 6个月
- [ ] 域名解析已指向阿里云ECS IP
- [ ] 域名隐私保护已关闭

## 备案流程

1. 购买阿里云ECS（中国大陆节点）
2. 登录阿里云备案系统：https://beian.aliyun.com/
3. 填写备案信息（主体信息 + 网站信息）
4. 上传材料 + 人脸核验
5. 阿里云初审（1-2个工作日）
6. 管局审核（上海管局通常7-15个工作日）
7. 备案成功，获得备案号
8. 在网站底部添加备案号并链接到工信部

## 备案后操作

1. 部署网站到ECS
2. 配置Nginx
3. 申请SSL证书（阿里云免费SSL）
4. 配置HTTPS
5. 在网站底部显示备案号：
   ```html
   <a href="https://beian.miit.gov.cn/" target="_blank">沪ICP备XXXXXXXX号</a>
   ```

## 关键注意事项

- **主体负责人必须是法定代表人**
- **域名持有者 ≠ 个人姓名**，必须为企业名称
- 备案期间网站需要可以访问（不能是空白页）
- 备案成功后，网站内容需与备案时描述一致
- 涉及"付费会员"属于经营性业务，严格来说需要**ICP经营许可证**（B25类），但初期可先备案，后续根据规模再申请

## 阿里云账号准备

需要在 GitHub Secrets 中添加：
- `ALIYUN_HOST` — ECS 公网IP
- `ALIYUN_USER` — SSH用户名（如 root）
- `ALIYUN_SSH_KEY` — SSH私钥
